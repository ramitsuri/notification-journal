# Notification Journal

Android, macOS and wearOS app for journaling.

## Features
- Share data between clients in real time.
  - Uses self hosted RabbitMq for desktop <-> Android
- On Android, it shows a persistent notification so that new journal entries can be added at any time.
- Privacy friendly - data doesn't leave the apps
- Light and dark themes

## Built using 
- Jetpack Compose 
- Kotlin Multiplatform
- RabbitMq
